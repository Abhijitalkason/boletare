⚙ API Layer — FastAPI
🔌
FastAPI Route Handler
api/kundli.py → compute_kundli(request: KundliRequest)
Receives the HTTP request. Parses birth_time string (HH:MM) into a datetime.time object using _parse_birth_time(). Maps birth_time_tier integer to BirthTimeTier enum. No DB, no auth required.
@router.post("/compute") _parse_birth_time() BirthTimeTier(tier)
calls pure function — no DB I/O
🔭 Computation Engine — Layer 1
🏗
Chart Orchestrator — Entry Point
engine/chart_computer.py → compute_birth_chart()
The single entry point for all chart computation. Coordinates 14 ordered steps across 6 sub-modules. Pure function — no side effects, no I/O.
1
date_to_jd()
engine/ephemeris.py
Converts birth date + time + timezone into a Julian Day number for Swiss Ephemeris.
2
check_boundaries()
engine/boundary.py
Dual-ephemeris scan within the birth time uncertainty window. Flags lagna_ambiguous and dasha_boundary_sensitive.
3
get_all_planet_longitudes()
engine/ephemeris.py
Swiss Ephemeris computes sidereal longitudes (arc-seconds) + retrograde flag for all 9 Vedic planets (Sun → Ketu).
4
compute_houses()
engine/houses.py
Placidus house cusps from lat/lon. Returns 12 HouseCusp objects + ascendant_arcsec + placidus_distorted flag.
5
compute_equal_houses_from_asc()
engine/houses.py
KP fallback: if Placidus distorted + ayanamsha=KP, switch to Equal House system. kp_on_equal_house flagged.
6
Chandra Lagna Failover
chart_computer.py (inline)
If lagna_ambiguous → use Moon's sign as ascendant + recompute Equal Houses from Moon position.
7
assign_planet_to_house()
engine/houses.py
Each planet's arc-second longitude is matched against house cusp boundaries to give house number 1–12.
8
compute_dignity()
engine/dignity.py
Classical dignity per planet: exaltation, debilitation, own sign, moolatrikona, friend, enemy, neutral. Returns dignity_score 0–1.
9
_build_planet_positions()
chart_computer.py
Assembles PlanetPosition objects: sign, sign_degrees, nakshatra, pada, house, dignity, is_retrograde for all 9 planets.
10
compute_dasha_tree()
engine/dasha.py
Vimshottari Dasha tree from Moon's nakshatra. If Moon near nakshatra boundary → alternate tree also computed.
11
compute_ashtakavarga()
engine/ashtakavarga.py
Ashtakavarga — Bindu Ashtakavarga (BAV) + Sarvashtakavarga (SAV) + Trikona Shodhana reduction.
12
compute_navamsha()
engine/navamsha.py
D-9 divisional chart. Each planet's navamsha sign computed, returned as separate PlanetPosition list.
13
QualityFlags()
chart_computer.py
Assembles trust metadata: birth_time_tier, lagna_mode, dasha_ambiguous, placidus_distorted, kp_on_equal_house.
14
BirthChart()
chart_computer.py
Immutable BirthChart object assembled with all data. Returned to kundli.py.
+
detect_all_yogas()
engine/yogas.py
Scans BirthChart for classical yogas: Raj Yoga, Gaj Kesari, Panch Mahapurusha, etc.
+
detect_all_doshas()
engine/doshas.py
Scans for Mangal Dosha, Kaal Sarp, Shani Dosha, etc. Also checks cancellation factors.
